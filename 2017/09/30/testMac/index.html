个人博客从Linux移植到Mac

2017-09-30
#Win7 #装系统 #简单高效
电脑日记

Note：当感到迷茫之时，仅要做的，就是咬紧牙关再坚持一下，胜利的曙光立刻就会来临。

一、环境安装

  1. node.js（在node.js官网中下载安装）node.js官网
  2. git（OS系统Store安装x-code就可以）
  3. 查看node和npm安装是否成功，返回版本即可

二、安装hexo

执行如下代码

sudo npm install hexo -g --no-optional

国内可能会用到淘宝源

set registry http://registry.npm.taobao.org/

如果执行如下代码，返回http://registry.npm.taobao.org/说明更换成功

npm get registry

如果安装过程中出现找不到package.json文件，执行下列代码：

三、卸载hexo

sudo npm uninstall hexo -g

四、设置git推送

1. 设置blog配置文件

  1. 打开blog文件夹下的_config.yml文件
  2. 找到最下方的type，输入git（注意冒号后面是带空格的）
  3. repo行可能没有，需要自己输入，后面跟上git上仓库中复制的http地址

2. 进入终端，输入git上传插件安装代码（安装时会提示输入git用户名及密码）

npm install hexo-deployer-git --save

3. 安装完毕后，输入获取代码

hexo g

4. 最后输入上传代码

hexo d

五、备注

Mac电脑还真是名不虚传，每次使用都能给人惊喜
