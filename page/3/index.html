1 Openwrt简介

Tp-link Wr703n是目前最小，最便宜的能够运行Linux的无线路由器。

主要配置：

Atheros 400MHz MIPS CPU

32MB RAM

4MB flash

10/100Mbps Ethernet interface

802.11b/g/n wireless interface with one internal antenna

USB 2.0 port

Micro-USB power socket, approximately 1W power draw

硬件的选型及组装

我们知道平衡车的机械设计，对于车模稳定工作是非常重要的。同理，恰当的硬件、单片机与传感器选型也是维持车模稳定运行的必要条件。本章概括性的叙述平衡车所用到的各个硬件单元及安装注意事项。

1 Arduino MEGA 2560 R3

Arduino Mega 2560 是基于Atmel公司生产的Atmega2560的一块最小系统板。他简单、易开发以及开源的特点，使得他在几年内风靡全球。例如，它的供电方式，不仅可以通过USB供电，也可以通过AC-DA电源适配器供电。

Microcontroller ATmega2560
Operating Voltage 5V
Input Voltage (recommended) 7-12V
Input Voltage (limit) 6-20V
Digital I/O Pins 54 (of which 15 provide PWM output)
Analog Input Pins 16
DC Current per I/O Pin 20 mA
DC Current for 3.3V Pin 50 mA
Flash Memory 256 KB of which 8 KB used by bootloader
SRAM 8 KB
EEPROM 4 KB
Clock Speed 16 MHz
Length 101.52 mm
Width 53.3 mm
Weight 37 g

2 MPU6050

MPU-6050是整合性6轴运动处理组件，相较于同时用陀螺仪和加速度器，免除了他们组合时出现的轴间差的问题，同时大大减少了芯片的包装空间。MPU-6050整合了3轴陀螺仪、3轴加速器，并含有一个数位运动处理硬件加速引擎DMP，然后由I2C通信协议将数据发送到单片机。

关于MPU6050我所接触的方法有两种：

  1. 通过I2C通信采集原始数据，然后通过Kalman滤波融合得到角度。
  2. 通过它自带的数位运动处理芯片DMP直接得到角度。

目前市场上流行一个串口MPU6050实际上是卖家在出售之前通过一个微型芯片已经做了相关滤波，并将其以串口的形式转发出来。实质上，和用户自己在单片机里面用卡尔曼滤波没有什么两样。

3 电机驱动L298N

L298N对于做智能车的人大多不陌生，他可以驱动两路直流电机，输出最大电流为2A，最高工作电压为46V，当然对于正常我们作比赛来说，大多说是7.4V或者是11.1V都属于安全电压。所以我们在设计过程中，要考虑到驱动的电流限制，以免出现烧毁驱动的情况。如果驱动需要更大电流的电机，那么L298不在适合，此时就需要自己设计驱动电路，这里就不在赘述。

5 光电码盘及编码器

光电编码器是一种体积小、使用简单、性能好和可靠性好的转速和位置检测装置，他在检测等领域应用十分广泛，编码器是码盘随位置的变化输出两路脉冲信号，然后我们根据这两路脉冲信号，判断出码盘旋转的方向并通过计数器对脉冲信号进行加减计数，通过固定时间得到的脉冲数，就可以得到电机转速.

光电码盘一个很重要的指标就是码盘的栅格个数。显而易见，栅格数越多精度越高，但在低速状态下会引入高频干扰，此时就需要通过软件进行低通滤波将其消除。

Win7游戏分辨率解决办法

发表于 2015-11-07   |   分类于 电脑日记   |  

Win7系统在玩游戏时，经常出现分辨率不正常的情况。比如，跑跑卡丁车、极品飞车、CS等游戏不能全屏显示，可以借鉴如下解决方法。

方法一、调整分辨率

1、调整桌面分辨率为1024X768，必须的，别的不行哦！
2、这时候我们发现仍旧没有全屏，windows把分辨率是改了，但是还站在中间的一小块。没关系，最关键的一步， 打开ATI设置中心，(或N卡的设置中心）你会惊喜的发现，全屏幕，居中，保持纵横比的选项能点了。OK，确定之后你会发现桌面已经全屏了。
3、现在，再把分辨率改回以前的就好了。

方法二、修改注册表

该方法和分辨率关系不大，只要修改一下注册表就行了，所有游戏一概全屏，情享受吧！
Win + R 出现运行命令框，输入注册表命令 regedit，依次选择如下：

右键Configuration，选择查找，输入Scaling，在右框找到scaling，右键scaling修改
将数值改为3即可，原值是4。
